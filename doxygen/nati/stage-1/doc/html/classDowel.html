Detailed Description

A Dowel is a cylindrical peace of wood. I have seeral Dowels as stock material. For production pieces are cut from the dowels. After a cut is done there exist two Dowels: The original one which is shorter now, a "new" Dowel to be used in production. This Dowels are cut in twobe used in production of some furniture All my Dowels have the same diameter, but the have differnt length.
